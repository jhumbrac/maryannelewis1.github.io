We offer:

  • Monogrammed stationery
  • Cards for any type of gathering

Birdsong Stationery can help you create the perfect wedding! From invitations to thank-you notes for gifts, Birdsong carefully crafts each note with your choice of decorative script and logos.

Special events are made for us? Are you looking for supercreative birthday invitations? We have them. What about a gift note for your bridesmaids? If you can come up with an idea, we can come up with the note.

And what about personalized stationery? We can create monogrammed envelopes or print addresses on writing paper or create a notepad with any message you'd like on it.
